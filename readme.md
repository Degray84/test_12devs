# Test project for 12devs

## Tech

- NodeJS | expressJS
- Redis | bullMQ | bull-board
- PostgreSQL | sequelize
- Nodemailer
- Swagger


## Installation

```sh
git clone https://github.com/Degray84/test_12devs.git test_12devs
cd test_12devs
npm i
npm run nodemon
```
